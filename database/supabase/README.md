# Database Supabase BIUperpus

Gunakan file `biuperpus_supabase.sql` untuk membuat database BIUperpus di Supabase.

Cara pakai:

1. Buka project Supabase.
2. Masuk ke menu SQL Editor.
3. Buat query baru.
4. Tempel isi `biuperpus_supabase.sql`.
5. Jalankan query.

File SQL ini berisi tabel utama aplikasi, relasi, index, dan data awal dari project lama.
